# Developer Hub Dynamic Plugin for OpenDORA

## Get Integrity SHA

[NPM Registry](https://registry.npmjs.org/@vikaspogu/developer-hub-open-dora)

```bash
npm info @vikaspogu/rhdh-dynamic-plugin-opendora 
```

## Build and Publish

```bash
yarn install
yarn tsc
yarn build
yarn run export-dynamic
yarn publish
```
